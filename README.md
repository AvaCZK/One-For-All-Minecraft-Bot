## Features
### Current
* Discord
    1. Pass to CMD Console
    2. Control & Forward
    3. Display Status
* Agent
    1. Task Scheduling
    2. Regular Restart (no longer necessary)
* DM Control
    1. Pass to CMD Console
* CMD Console
    1. Identify Command Source (DM / DC / Console)
